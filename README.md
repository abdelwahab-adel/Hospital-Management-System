# 📖 Overview

Hospital Management System is a full-featured web application developed using **Laravel** to help hospitals and clinics automate their daily operations.

The system provides an efficient environment for managing patients, doctors, appointments, departments, medical records, billing, and administrative tasks while ensuring data security, scalability, and high performance.

It is designed following modern software engineering principles and Laravel best practices to ensure maintainability, clean architecture, and future scalability.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/abdelwahab-adel/Hospital-Management-System.git
````

Move to project

```bash
cd Hospital-Management-System
```

Install dependencies

```bash
composer install
```

Copy environment file

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Configure your database inside the `.env` file.

Run migrations

```bash
php artisan migrate
```

(Optional)

```bash
php artisan db:seed
```

Start development server

```bash
php artisan serve
```

Open

```
http://127.0.0.1:8000
```
